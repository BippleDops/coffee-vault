# Coffee Goals Folder

This folder contains **Coffee Goals** (`type: coffee-goal`) - personal development objectives, learning goals, exploration targets, and achievement milestones.

## Purpose

Track personal coffee development:
- Learning goals (methods, techniques, courses)
- Exploration goals (origins, beans to try)
- Achievement targets (ratings, certifications)
- Equipment purchase goals
- Coffee travel plans

## Usage

Create goals using: `Templates/Coffee Goal.md`

## File Naming

Use descriptive names: `goal-title.md` or `goal-type-title.md`

## Linking

Link goals to:
- Coffee Logs (`related-logs` property)
- Bean Profiles (`related-beans` property)
- Guides and References (`related-guides` property)
- Equipment Models (`equipment-to-purchase` property)
