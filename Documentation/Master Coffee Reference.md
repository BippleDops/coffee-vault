### Grinder Guide

**Grinder Hierarchy**:

**Budget** ($50-150):
- Hand grinders (Hario, Timemore)
- Entry electric blade (avoid for specialty)
- Limitations: Inconsistent, slow

**Mid-Range** ($150-400):
- Baratza Encore (drip, pour-over)
- Fellow Ode (filter coffee)
- Decent for everything except espresso

**Espresso-Capable** ($400-800):
- Baratza Sette 270
- Eureka Mignon
- Can do espresso + filter

**High-End** ($800-2000+):
- Niche Zero (single-dose)
- Fellow Ode + SSP burrs
- Lagom P64
- Professional-grade consistency

**Ultra** ($2000+):
- EK43 (legendary, commercial)
- Monolith
- Professional commercial grinders
- Competition level

**Key Principle**: Spend more on grinder than brewer!

**Your Coffee Journey**:
```datacore
const logs = dc.pages('"Coffee Logs"');
const totalSessions = logs.length;
const avgRating = logs.length > 0 ? (logs.sum(p => p.rating) / logs.length).toFixed(2) : 0;
const totalCups = logs.sum(p => p["cups-brewed"] || 0);

return <div>
  <h3>Your Coffee Journey</h3>
  <p>üìä {totalSessions} sessions logged</p>
  <p>‚≠ê {avgRating}/5 average rating</p>
  <p>‚òï {totalCups} total cups consumed</p>
</div>
```

**Favorite Origin**:
```datacore
const byOrigin = {};
dc.pages('"Coffee Logs"').forEach(log => {
  if (!log.origin) return;
  if (!byOrigin[log.origin]) byOrigin[log.origin] = {count: 0, totalRating: 0};
  byOrigin[log.origin].count++;
  byOrigin[log.origin].totalRating += log.rating || 0;
});

const sorted = Object.entries(byOrigin)
  .map(([origin, stats]) => ({
    origin,
    sessions: stats.count,
    avgRating: (stats.totalRating / stats.count).toFixed(2)
  }))
  .sort((a, b) => b.sessions - a.sessions);

return sorted.length > 0 ? (
  <div>
    <p><strong>Your most-logged origin:</strong> {sorted[0].origin} ({sorted[0].sessions} sessions, {sorted[0].avgRating} avg rating)</p>
  </div>
) : <p>Log more coffee to discover patterns!</p>
```
