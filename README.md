## What is Coffee Vault?

An intelligent coffee tracking system that combines:

- **Data Tracking** - Log every brew with detailed parameters
- **Machine Learning** - Get quality predictions and optimization suggestions
- **Analytics** - 8 ML-powered dashboards for insights
- **Automation** - 22 scripts for reports, inventory, and batch operations
- **Visualizations** - 17 interactive tools for exploration
- **Education** - Scientific references and brewing guides

## System Requirements

**Required**:
- Obsidian 1.4.0+
- Datacore plugin
- Templater plugin

**Optional**:
- Node.js 16+ (for automation scripts)
- Modern web browser (for visualizations)
- Calendar plugin
- Tasks plugin

---

## First Steps

1. Install Datacore + Templater plugins
2. Create your first coffee log using `Templates/Coffee-Log-v3.md`
3. After 5 logs, explore `Analytics/1-Monthly-Analytics-Dashboard.md`
4. Try visualizations in `Visualizations/` folder
5. Read documentation as needed

## Features by Skill Level

**Beginner** (1-20 logs):
- Basic statistics and tracking
- Consistency analysis
- Simple visualizations
- Flavor compass, grind calculator

**Intermediate** (20-50 logs):
- Correlation analysis
- Trend detection
- Method optimization
- Cost-performance tracking

**Advanced** (50+ logs):
- ML quality predictions
- K-Means clustering
- Anomaly detection
- Full automation suite

## Technology Stack

- **Platform**: Obsidian.md (local-first knowledge management)
- **Query Engine**: Datacore plugin
- **Templates**: Templater plugin
- **Automation**: JavaScript (Node.js, CommonJS)
- **Visualizations**: HTML5 Canvas, vanilla JavaScript
- **ML Algorithms**: KNN, K-Means, Decision Trees (pure JS)
- **Statistics**: 25+ functions (correlation, regression, outliers)
