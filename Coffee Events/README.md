# Coffee Events Folder

This folder contains **Coffee Events** (`type: coffee-event`) - coffee experiences beyond brewing including cafe visits, classes, competitions, festivals, and travel.

## Purpose

Document coffee experiences:
- Cafe visits and reviews
- Coffee classes and workshops
- Competition participation
- Festival and event experiences
- Roastery visits
- Coffee travel experiences

## Usage

Create events using: `Templates/Coffee Event.md`

## File Naming

Use date-name format: `YYYY-MM-DD-event-name.md` or descriptive names

## Linking

Link events to:
- Roaster Profiles (roastery visits)
- Bean Profiles (beans discovered/tried)
- Coffee Logs (brews during events)
- Cupping Sessions (cupping events)
