# Cupping Sessions Folder

This folder contains **Cupping Sessions** (`type: cupping-session`) - formal cupping evaluations, comparative tastings, and professional cupping events.

## Purpose

Document formal cupping activities:
- SCA/COE/CQI protocol cupping sessions
- Comparative coffee evaluations
- Blind cupping sessions
- Professional cupping events
- Training and educational cuppings

## Usage

Create cupping sessions using: `Templates/Cupping Session.md`

## File Naming

Use date-based naming: `YYYY-MM-DD-Cupping-Session.md` or descriptive names for special events

## Linking

Link cupping sessions to:
- Bean Profiles (`samples` property)
- Coffee Logs (related sessions)
- Coffee Events (if part of an event)
