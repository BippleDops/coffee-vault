## How to Set Up Custom Shortcuts

Coffee Vault shortcuts require manual setup in Obsidian. Follow these steps:

### Step 1: Open Hotkeys Settings

1. Open Obsidian Settings (`Cmd/Ctrl + ,`)
2. Navigate to **Settings â†’ Hotkeys**
3. Search for the command you want to bind

### Step 2: Set Up Template Shortcuts

For template shortcuts, you need to create "Template Hotkeys":

1. Install the **Templater** plugin (if not already installed)
2. In Settings â†’ Templater â†’ Hotkeys
3. Add a new hotkey for each template:
   - **Quick Log**: `Cmd/Ctrl + Shift + L` â†’ `Templates/Coffee-Log-Quick.md`
   - **Standard Log**: `Alt + 2` â†’ `Templates/Coffee-Log-Standard.md`
   - **Detailed Log**: `Alt + 3` â†’ `Templates/Coffee-Log-Detailed.md`

### Step 3: Set Up Script Shortcuts

For data extraction and automation:

1. Search for "Execute Code" or "Run JavaScript" in Hotkeys
2. Bind `Cmd/Ctrl + Shift + E` to data extraction script
3. Bind `Cmd/Ctrl + Shift + A` to automation panel

### Step 4: Set Up Navigation Shortcuts

For dashboard navigation:

1. Search for "Open file" in Hotkeys
2. Set up these bindings:
   - `Alt + H` â†’ Open `HOME-DASHBOARD.md`
   - `Alt + N` â†’ Open `Views/Persona-NOVICE-Dashboard.md`
   - `Alt + E` â†’ Open `Views/Persona-ENTHUSIAST-Dashboard.md`
   - `Alt + P` â†’ Open `Views/Persona-PROFESSIONAL-Dashboard.md`

### Step 5: Test Your Shortcuts

After setting up:
1. Close settings
2. Try each shortcut to verify it works
3. Adjust as needed for your workflow

## Tips for Learning Shortcuts

### Progressive Learning Approach

**Week 1**: Master the essentials (ğŸ”´)
- Focus on `Cmd/Ctrl + O`, `Cmd/Ctrl + P`, and `Shift + Space`
- Practice daily until they become muscle memory

**Week 2**: Add useful shortcuts (ğŸŸ¡)
- Integrate persona switching (`Alt + N/E/P`)
- Learn template shortcuts (`Alt + 1/2/3`)

**Week 3+**: Explore advanced features (ğŸŸ¢)
- Add data/analytics shortcuts as needed
- Customize visualization controls to your workflow

### Practice Exercises

1. **Speed Logging**: Try logging a coffee brew using only keyboard shortcuts
2. **Dashboard Tour**: Navigate through all persona dashboards without using mouse
3. **Template Race**: Create 3 different logs using template shortcuts
4. **Viz Master**: Open a visualization and control it entirely with keyboard

## Customization Tips

### Creating Your Own Shortcuts

**Best Practices**:
1. Use consistent patterns (e.g., all templates use `Alt + Number`)
2. Avoid conflicts with Obsidian defaults
3. Group related actions (e.g., all data actions use `Cmd/Ctrl + Shift`)
4. Keep frequently-used shortcuts easy to reach

**Popular Customizations**:
- `Cmd/Ctrl + J` - Daily journal entry
- `Cmd/Ctrl + Shift + C` - Copy current file path
- `Cmd/Ctrl + Shift + X` - Export to PDF

### Platform Differences

**macOS**:
- Use `Cmd` for most shortcuts
- `Option` = `Alt`
- `Control` rarely used

**Windows/Linux**:
- Use `Ctrl` for most shortcuts
- `Alt` works as shown
- Some shortcuts may conflict with system shortcuts

## Accessibility Notes

### For Screen Reader Users

All shortcuts are screen reader compatible. Navigate using:
- `Tab` - Move between interactive elements
- `Enter` - Activate buttons/links
- `Space` - Toggle checkboxes
- `Escape` - Close dialogs

### For Low Vision Users

Increase shortcut visibility:
1. Settings â†’ Appearance â†’ Font size
2. Enable "Show hotkey hints" in Settings
3. Use high contrast theme for better visibility

### For Motor Impairments

Alternatives to keyboard shortcuts:
- Use command palette (`Cmd/Ctrl + P`) for all actions
- Enable sticky keys in OS settings
- Use voice control (macOS Voice Control, Windows Speech Recognition)

## Troubleshooting

### Shortcut Not Working?

1. **Check for conflicts**: Settings â†’ Hotkeys â†’ Search for shortcut
2. **Verify plugin is enabled**: Settings â†’ Community Plugins
3. **Restart Obsidian**: Sometimes required after changing shortcuts
4. **Check focus**: Some shortcuts only work in specific contexts

### Common Conflicts

|----------|------------------|--------------|----------|
