## How to Use This Board

### Goal Stages

1. **Planned**: Goals set but not yet started
2. **In Progress**: Actively working on
3. **On Hold**: Paused temporarily (with reason)
4. **Completed**: Achieved! Move here to celebrate
5. **Achievements**: Major milestones and celebrations
6. **Quarterly Review**: Track quarterly objectives

### Goal Format

```markdown
- [ ] Goal Name #goal:type #priority
  - Progress: X% or X/Y
  - Current status or challenge
  - Next steps or milestones
  - Link: [[Coffee Goal]]
```

### Goal Types

- `#goal:method` - Learning brewing methods
- `#goal:skill` - Specific skills (cupping, latte art)
- `#goal:exploration` - Try origins, beans, roasters
- `#goal:quality` - Achieve rating targets
- `#goal:equipment` - Purchase equipment
- `#goal:certification` - Formal certifications
- `#goal:travel` - Origin visits, coffee travel

### Priority Levels

- `#high-priority` - Focus on these
- `#medium-priority` - Work on when possible
- `#low-priority` - Someday/maybe
- `#active` - Currently main focus

### Linking

**Goal Profiles**: Create formal goal with `Templates/Coffee Goal.md`  
**Coffee Logs**: Link to practice sessions  
**Resources**: Link to guides, references  
**Equipment**: Link to equipment profiles

### Progress Tracking

Update cards weekly:
- Move between columns as status changes
- Update progress percentages
- Note milestones achieved
- Adjust timelines as needed

### Celebrating Achievements

When completing a goal:
1. Move to **Completed** column
2. Add completion date
3. Note what you learned
4. Set next related goal
5. Celebrate! (Special coffee, share achievement)
