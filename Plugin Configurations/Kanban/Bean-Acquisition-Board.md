## How to Use This Board

### Getting Started
1. Remove the example cards above (they're just for reference)
2. Start adding your real beans
3. Move cards through the workflow as you progress

### Card Format
```markdown
- [ ] Bean Name #labels
  - Key details
  - Link: [[Related Note]]
  - Notes and observations
```

### Labels to Use
- Origin: `#single-origin` `#blend`
- Price: `#budget` `#medium-price` `#premium`
- Roast: `#light-roast` `#medium-roast` `#dark-roast`
- Purpose: `#espresso` `#pour-over` `#cold-brew`
- Priority: `#must-try` `#interested` `#someday`

### Workflow
1. **Research**: Heard about, investigating
2. **Purchase Planned**: Research done, will buy
3. **In Transit**: Ordered, tracking delivery
4. **Testing**: Arrived, first 3-5 brews
5. **In Rotation**: Active use, daily/weekly
6. **Finished**: Bag empty, review complete

### Linking to Vault
- Create Bean Profile notes for beans in testing
- Link cards to profiles: `[[Bean Name]]`
- Reference Coffee Logs in card notes
- Link to roaster profiles

### Tips
- Add roast dates to track freshness
- Note grind settings that work well
- Track brewing methods used
- Document flavor evolution
- Set reminders for reorder dates
- Archive Finished cards regularly

### Archive Settings
Completed cards can be archived:
- Right-click card → Archive
- Keeps board clean
- Preserves history
- Can restore if needed
