## Board Instructions

### Cupping Stages

1. **Planned Cuppings**: Sessions designed but not scheduled
2. **This Week**: Scheduled for next 7 days
3. **Completed**: Cupping sessions finished
4. **Skills to Develop**: Sensory abilities to work on
5. **Cupping Goals**: Formal goals around cupping
6. **Cupping Schedule**: Regular practice calendar

### Card Format

```markdown
- [ ] Cupping Name #cupping #tags
  - Coffees: List of coffees to cup
  - Purpose: Why doing this cupping
  - Date: When scheduled
  - Protocol: SCA, informal, blind, etc.
  - Link: [[Cupping Sessions/Session-Name]]
```

### Cupping Types

- `#comparison` - Side-by-side comparison
- `#blind` - Blind cupping (unknown samples)
- `#training` - Practice and skill development
- `#origin-comparison` - Compare origins
- `#processing-comparison` - Compare processing methods
- `#roast-comparison` - Compare roast levels

### Linking

**Cupping Sessions**: Create with `Templates/Cupping Session.md`  
**Bean Profiles**: Link to beans being cupped  
**Coffee Goals**: Link to sensory development goals  
**Scientific References**: Link to cupping protocols

### Best Practices

**Preparation**:
- Schedule cuppings when palate is fresh (morning)
- Prepare identical sample sizes (12g standard)
- Use consistent water and temperature
- Print SCA cupping forms

**Execution**:
- Follow protocol systematically
- Cup hot, warm, and cool
- Take detailed notes
- Score independently before comparing

**Post-Cupping**:
- Create cupping session note
- Link to bean profiles
- Update preferences based on learnings
- Track scores for consistency

**Frequency**:
- Weekly practice builds skill
- Monthly comparisons deepen understanding
- Quarterly formal evaluations track progress
