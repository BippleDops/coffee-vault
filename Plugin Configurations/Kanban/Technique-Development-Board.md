## How to Use This Board

### Getting Started
1. Remove example cards
2. List techniques you want to learn
3. Move cards as you progress through stages
4. Document mastered techniques in Brewing Guides

### Card Format
```markdown
- [ ] Technique Name #method #skill-level
  - Learning resources
  - Practice notes
  - Progress tracking
  - Link: [[Related Guide]]
```

### Labels to Use

**Method**:
- `#pour-over` - V60, Chemex, Kalita
- `#espresso` - Espresso techniques
- `#french-press` - French press methods
- `#aeropress` - AeroPress techniques
- `#cold-brew` - Cold brew methods
- `#general` - Universal techniques

**Skill Level**:
- `#beginner` - Foundational skills
- `#intermediate` - Building on basics
- `#advanced` - Expert techniques
- `#expert` - Competition-level

**Focus Area**:
- `#consistency` - Repeatability
- `#precision` - Accuracy and control
- `#flavor` - Taste optimization
- `#equipment` - Tool mastery
- `#theory` - Understanding principles

**Time Investment**:
- `#quick` - 1-2 weeks to learn
- `#medium` - 1-2 months to learn
- `#long` - 3+ months to learn

### Learning Stages

#### 1. To Learn
Techniques you want to develop:
- List anything interesting
- No commitment yet
- Prioritize when ready
- Research resources

#### 2. Learning
Active study phase:
- Reading articles/books
- Watching videos
- Understanding theory
- Taking notes
- Not yet practicing

**What to track**:
- Resources being used
- Key concepts learned
- Questions to explore
- When to start practice

#### 3. Practice
Deliberate practice phase:
- Applying what you learned
- Focused repetition
- Tracking attempts
- Noting improvements
- Identifying challenges

**Practice schedule**:
- Set frequency (daily, 3x/week, etc.)
- Track each session
- Document in Coffee Logs
- Review progress weekly
- Adjust approach as needed

**What to track**:
- Number of attempts
- Success rate
- Common mistakes
- Breakthrough moments
- Resource links

#### 4. Refinement
Improving consistency:
- Good results most of the time
- Working on edge cases
- Reducing variance
- Optimizing process
- Fine-tuning

**Refinement goals**:
- Increase success rate to 90%+
- Reduce time to execute
- Improve efficiency
- Handle variations (beans, conditions)
- Troubleshoot quickly

**What to track**:
- Current success rate
- Specific areas to improve
- Experiments tried
- Results of changes
- Target metrics

#### 5. Mastered
Consistent excellence:
- Can execute without thinking
- Reliable results
- Understand why it works
- Can adapt to conditions
- Confident in skill

**Mastery criteria**:
- 90%+ success rate
- Can explain to others
- Can troubleshoot issues
- Documented in guide
- Repeatable over time

#### 6. Teaching
Sharing knowledge:
- Teaching friends
- Creating guides
- Making videos
- Writing articles
- Mentoring others

**Teaching activities**:
- One-on-one instruction
- Written guides
- Video tutorials
- Blog posts
- Workshops/classes

### Technique Categories

**Foundational Techniques** (Start here):
- Water temperature control
- Grind size adjustment
- Ratio calculation
- Timing methods
- Tasting and evaluation

**Pour-Over Techniques**:
- Bloom technique
- Pour patterns (spiral, center, pulse)
- Flow rate control
- Agitation methods
- Drawdown management

**Espresso Techniques**:
- Dosing consistency
- Distribution (WDT, etc.)
- Tamping pressure and levelness
- Pre-infusion timing
- Pressure profiling
- Milk steaming and latte art

**Extraction Theory**:
- Understanding TDS and extraction yield
- Identifying under/over extraction
- Adjusting variables systematically
- Taste mapping
- Recipe development

**Advanced Skills**:
- Cupping protocols
- Sensory training
- Roast development (if home roasting)
- Water chemistry
- Competition techniques

### Practice Strategies

**Deliberate Practice**:
1. Identify specific skill to improve
2. Set measurable goal
3. Practice with focus
4. Get feedback (taste, timing, appearance)
5. Adjust and repeat
6. Document progress

**Side-by-Side Comparison**:
- Make same recipe twice
- Change ONE variable
- Compare results
- Note differences
- Choose better approach

**Video Recording**:
- Record your technique
- Watch and analyze
- Compare to expert videos
- Identify improvements
- Track evolution over time

**Tasting Training**:
- Regular cupping sessions
- Flavor wheel practice
- Blind comparisons
- Note-taking discipline
- Developing palate

### Documentation

Create guides for mastered techniques:
- Step-by-step instructions
- Key parameters
- Common mistakes
- Troubleshooting tips
- Photos or diagrams
- Video links
- Equipment needed
- Time investment

Link to:
- Brewing Guides (your technique docs)
- Coffee Logs (practice sessions)
- Equipment notes (tools used)
- Bean profiles (beans that worked well)

### Progress Tracking

**Weekly Review**:
- What techniques practiced?
- What improved?
- What challenged you?
- What to focus on next week?

**Monthly Assessment**:
- Ready to move cards forward?
- New techniques to add?
- Any to pause/archive?
- Celebrate progress!

**Quarterly Goals**:
- Master 1-2 techniques per quarter
- One beginner, one intermediate/advanced
- Document in Brewing Guides
- Share with community

### Tips for Effective Learning

**Start Simple**:
- Master basics before advanced
- One technique at a time
- Build solid foundation
- Don't rush

**Be Patient**:
- Skill development takes time
- Plateaus are normal
- Keep practicing
- Trust the process

**Get Feedback**:
- Taste your coffee critically
- Ask others for opinions
- Compare to reference brews
- Use measurement tools

**Document Everything**:
- Keep detailed Coffee Logs
- Note what worked/didn't
- Track parameters
- Review regularly

**Embrace Failure**:
- Bad brews teach lessons
- Troubleshoot systematically
- Experiment fearlessly
- Learn from mistakes

**Stay Curious**:
- Always ask "why?"
- Research deeply
- Try new approaches
- Challenge assumptions

Build a resource collection:

- Courses and classes
- Expert contacts
- Coffee communities
