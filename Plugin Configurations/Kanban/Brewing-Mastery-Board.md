## Board Instructions

### Workflow Stages

1. **To Learn**: Methods and skills you want to acquire
2. **In Practice**: Currently working on, practicing regularly
3. **Mastered**: Achieved consistent excellence (10+ brews, 85%+ success rate)
4. **Goals Set**: Formal goals created and tracked
5. **Resources Needed**: Equipment, books, courses needed for learning

### Card Management

**Creating Cards**:
- Use checkbox format: `- [ ] Card Title #tags`
- Add details in bullet points below
- Link to relevant vault notes: `[[Note Name]]`
- Tag appropriately for filtering

**Moving Cards**:
- Drag between columns as you progress
- Archive when fully mastered or goal achieved
- Keep board current and actionable

**Tagging**:
- Method: `#method:v60`, `#method:espresso`
- Level: `#beginner`, `#intermediate`, `#advanced`
- Type: `#skill:cupping`, `#skill:latte-art`
- Priority: `#high-priority`, `#medium-priority`

### Integration with Vault

**Link to Goals**: `[[Coffee Goals/Goal-Name]]`  
**Link to Guides**: `[[Brewing Guides/Method-Guide]]`  
**Link to Logs**: Reference specific successful brews  
**Track Progress**: Update cards with achievements

### Best Practices

- Move cards weekly based on progress
- Set specific, measurable mastery criteria
- Link all resources and related notes
- Archive completed items monthly
- Review board every Sunday
- Celebrate mastery achievements!
