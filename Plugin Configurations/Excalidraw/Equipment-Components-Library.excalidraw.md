# Equipment Components Library

This is a component library for coffee equipment. Copy and paste components into your diagrams.

## Contents
- Kettles (Gooseneck, Standard, Electric)
- Grinders (Hand, Burr, Blade)
- Scales (Basic, Precision)
- Brewers (V60, Chemex, French Press, AeroPress, Espresso Machine)
- Cups and Vessels
- Accessories (Timer, Thermometer, Spoons)

## Usage Instructions

### To use components:
1. Open this file in Excalidraw view
2. Select the component(s) you want (Ctrl/Cmd + click for multiple)
3. Copy (Ctrl/Cmd + C)
4. Open your target drawing
5. Paste (Ctrl/Cmd + V)
6. Resize and arrange as needed

### Component Styling
All components use standardized colors:
- Coffee brown (#8B4513) for equipment bodies
- Black (#000000) for outlines
- Gray (#95A5A6) for metal parts
- Blue (#4A90E2) for water elements

### Drawing Tips
- Components are drawn at approximate 1:10 scale
- Maintain proportions when resizing
- Group related components together
- Use consistent line weights (2px for outlines)

## Component Descriptions

### Kettles
**Gooseneck Kettle**: Precision pouring, pour-over methods
**Standard Kettle**: General purpose, French press, large volumes
**Electric Kettle**: Temperature control, consistent heating

### Grinders
**Hand Grinder**: Portable, manual control, quiet
**Burr Grinder**: Consistent grind, adjustable settings
**Blade Grinder**: Basic, budget option

### Brewers
**V60 Dripper**: Cone shape, spiral ridges, single-cup
**Chemex**: Hourglass, thick filters, 3-8 cups
**French Press**: Plunger, full immersion, 4-8 cups
**AeroPress**: Cylinder, pressure brewing, 1-3 cups
**Espresso Machine**: High pressure, milk steaming

### Scales
**Basic Scale**: Weight only, 1g precision
**Precision Scale**: Weight + timer, 0.1g precision

### Accessories
**Timer**: Countdown/stopwatch
**Thermometer**: Temperature measurement
**Dosing Spoon**: Measurement, stirring
