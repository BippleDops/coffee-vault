# Pour Over Setup Template

A complete pour-over brewing station layout template.

## Template Contents
- Scale with dripper placement
- Kettle positioning
- Server/mug placement
- Parameter annotation boxes
- Timer indicator
- Water flow direction arrows

## How to Use This Template

### Quick Start
1. Duplicate this file: Right-click → Duplicate
2. Rename to your specific setup (e.g., "V60-Morning-Setup")
3. Open in Excalidraw view
4. Customize components and parameters
5. Link to your Coffee Log or Brewing Guide

### Customization Guide

**Replace Components**:
- Swap dripper type (V60, Kalita Wave, Bee House)
- Change kettle style
- Adjust cup/server size
- Add/remove accessories

**Annotate Parameters**:
- Coffee dose (grams)
- Water temperature (°F)
- Total brew time (minutes)
- Ratio (1:X format)
- Grind size
- Pour pattern (number of pours)

**Document Flow**:
- Number pour sequences
- Show water flow direction
- Indicate bloom phase
- Mark pulse timing

## Standard Layout

```
[Kettle]  →  [Scale with Dripper]  →  [Server/Cup]
                    ↓
              [Parameter Box]
              - Dose: ___ g
              - Water: ___ g
              - Ratio: 1:___
              - Temp: ___ °F
              - Time: ___ min
              - Grind: ___

[Timer]     [Notes]
```
