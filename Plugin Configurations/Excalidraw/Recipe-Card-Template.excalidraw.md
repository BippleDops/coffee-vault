# Visual Recipe Card Template

Create beautiful, illustrated brewing recipes with this template.

## Template Structure

```
┌─────────────────────────────────────────────────┐
│  ☕ [RECIPE NAME]                                │
│  Bean: [Name] | Method: [Type]                  │
├─────────────────────────────────────────────────┤
│  EQUIPMENT                 │  PARAMETERS         │
│  [Equipment diagram]       │  Dose: ___ g        │
│                           │  Water: ___ g       │
│                           │  Temp: ___ °F       │
│                           │  Ratio: 1:___       │
│                           │  Time: ___ min      │
│                           │  Grind: ___         │
├─────────────────────────────────────────────────┤
│  STEPS                                          │
│  1. [Step diagram with annotation]             │
│  2. [Step diagram with annotation]             │
│  3. [Step diagram with annotation]             │
│  4. [Step diagram with annotation]             │
├─────────────────────────────────────────────────┤
│  NOTES & TIPS                                   │
│  • [Key observation]                           │
│  • [Pro tip]                                   │
│  • [Common mistake to avoid]                   │
└─────────────────────────────────────────────────┘
```

## How to Create a Recipe Card

### Step 1: Add Header Information
- Recipe name with emoji
- Bean name/origin
- Brewing method
- Date created
- Skill level (beginner/intermediate/advanced)

### Step 2: Draw Equipment Section
Use components from Equipment-Components-Library:
- Copy relevant equipment
- Arrange left to right
- Scale appropriately
- Label each piece

### Step 3: Fill Parameter Box
Include all critical parameters:
- Coffee dose (grams)
- Water amount (grams)
- Ratio (1:X format)
- Water temperature (°F)
- Total brew time (minutes:seconds)
- Grind size (text description)
- Yield (grams or ml)

### Step 4: Illustrate Steps
For each brewing step:
- Draw simple visual representation
- Add timing annotation
- Show water flow or action
- Note technique details

Example steps for pour-over:
1. Rinse filter, add grounds
2. Bloom pour (0:00-0:45)
3. Main pour(s) with pattern
4. Drawdown and serve

### Step 5: Add Notes Section
Include:
- What to look for (cues)
- Pro tips for success
- Common mistakes
- Flavor expectations
- Troubleshooting hints

## Parameter Reference

### Coffee Dose
- Single cup: 15-18g
- Double cup: 25-30g
- Full carafe: 40-60g

### Ratios
- Standard: 1:15-1:17
- Strong: 1:13-1:15
- Mild: 1:17-1:20

### Water Temperature
- Light roast: 200-205°F
- Medium roast: 195-200°F
- Dark roast: 190-195°F

### Grind Size
- Extra-fine: Turkish, espresso
- Fine: Espresso, moka pot
- Medium-fine: Pour-over, drip
- Medium: Drip, Chemex
- Medium-coarse: Clever, Kalita
- Coarse: French press, cold brew

### Brew Time
- Espresso: 25-30 seconds
- Pour-over: 2:30-4:00 minutes
- French press: 4:00 minutes
- AeroPress: 1:30-2:30 minutes

## Styling Guide

**Title**: 24pt, bold, coffee brown (#8B4513)
**Section Headers**: 18pt, bold, black
**Body Text**: 14pt, regular, dark gray
**Annotations**: 12pt, italic, medium gray

- Equipment: Coffee brown (#8B4513)

- Border: 2px, solid, black

## Tips for Great Recipe Cards

1. **Keep it Visual**: More diagrams, less text
2. **Be Specific**: Exact numbers, not ranges
3. **Show Flow**: Use arrows for process
4. **Highlight Critical Steps**: Bold or color
5. **Test First**: Brew the recipe before documenting
6. **Iterate**: Update based on results
7. **Credit Sources**: Note recipe origins
8. **Date Versions**: Track recipe evolution
