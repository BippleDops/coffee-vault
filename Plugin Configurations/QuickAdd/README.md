### Quick Coffee Log

**Macro Name**: "Quick Coffee"  
**Template**: `Templates/Quick Coffee Capture.md`  
**Folder**: `Coffee Logs/`  
**Purpose**: Fast logging with minimal fields

**Prompts**:
1. Beans name
2. Brew method
3. Rating (1-5)
4. Quick notes

**Result**: Creates coffee log with essentials, skip detailed fields for speed

### Full Coffee Log

**Macro Name**: "Full Coffee Log"  
**Template**: `Templates/Coffee-Log-v3.md`  
**Folder**: `Coffee Logs/`  
**Purpose**: Complete logging with all parameters

**Prompts**:
1. Beans
2. Brew method
3. Grind size
4. Dose/water
5. Temperature
6. Rating
7. Tasting notes
