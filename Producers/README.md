# Producers Folder

This folder contains **Producer/Farm Profiles** (`type: producer-profile`) - comprehensive documentation of coffee producers, farms, cooperatives, washing stations, exporters, and importers.

## Purpose

Track complete supply chain information:
- Producer relationships and direct trade connections
- Certifications and sustainability metrics
- Farm characteristics and production capacity
- Quality profiles and consistency
- Personal relationships and visits

## Usage

Create producer profiles using: `Templates/Producer Profile.md`

## File Naming

Use kebab-case naming: `cooperative-name.md` or `farm-name.md`

## Linking

Link producers to:
- Bean Profiles (`producer-link` property)
- Roaster Profiles (`roaster-partners` property)
- Origin Profiles (via country/region)
