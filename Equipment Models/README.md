# Equipment Models Folder

This folder contains **Equipment Models** (`type: equipment-model`) - comprehensive equipment database with specifications, pricing, reviews, and compatibility information.

## Purpose

Research and document equipment:
- Equipment specifications database
- Pricing and availability tracking
- Community reviews and ratings
- Model comparisons
- Compatibility information

## Usage

Create equipment models using: `Templates/Equipment Model.md`

## File Naming

Use brand-model format: `brand-model-name.md`

## Linking

Link equipment models to:
- Equipment Profiles (user-owned equipment)
- Recipe Profiles (required equipment)
- Coffee Logs (equipment used)
