## üéì Tips & Techniques

### Getting Best Results

**Optimizing Auto-Start**:
- **Sensitivity**: Start with Medium (default), adjust if false triggers
- **False triggers**: Switch to Low sensitivity (5g threshold)
- **Missed starts**: Switch to High sensitivity (1g threshold)
- **Consistent technique**: Pour same way each time (reliable triggering)

**Using Ratio Calculator**:
- **Experimentation**: Try 1:15 (strong), 1:16 (balanced), 1:17 (light)
- **Taste adjust**: Sour = increase ratio (1:17‚Üí1:18), Bitter = decrease (1:16‚Üí1:15)
- **Custom ratios**: Find your sweet spot (e.g., 1:16.5 via manual calculation)
- **Record keeping**: Note which ratio for which coffee (manual log)

**Visual Progress Tracking**:
- **Progress bar**: Watch LED bar fill (visual brew pace)
- **Target weight**: Monitor "current / target" (e.g., "180 / 250g")
- **Countdown**: When close, watch "-10g" countdown (slow pour to hit exact)
- **Alert**: Listen for beep (target reached, can pour beyond if desired)

**Battery Life Extension**:
- **Auto-shutoff**: Enable 3-minute shutoff (saves 40% battery)
- **Dimmed mode**: Use dimmed brightness (saves 20% battery)
- **Power off**: Manually turn off after use (don't rely on auto-shutoff)
- **Rechargeable AAA**: Use NiMH rechargeable (environmental, cost-effective)

### Advanced Techniques

**Ratio Experimentation**:
- **Light roasts**: Try 1:17-1:18 (extract gently, highlight brightness)
- **Dark roasts**: Try 1:14-1:15 (concentrated, rich, avoid over-extraction)
- **Medium roasts**: 1:16 standard (balanced)
- **Cold brew**: 1:5-1:8 concentrate (dilute 1:1 for drinking)

**Multi-Stage Pouring** (manual tracking):
- **Bloom**: 2√ó coffee weight (e.g., 15g coffee = 30g water)
- **First pour**: 60% total (e.g., 250g total = 150g first pour)
- **Second pour**: Remaining 40% (100g)
- **Note times**: Manually track bloom end (0:45), first pour end (1:30), etc.

**French Press Variations**:
- **Short steep**: 3:00 for brighter, lighter body
- **Standard**: 4:00 classic
- **Long steep**: 5:00-6:00 for full extraction, heavier body
- **Iced French press**: Brew concentrate (1:12), pour over ice

**Troubleshooting**:

**Scale won't power on**:
- Check batteries (replace with fresh AAA)
- Ensure batteries inserted correctly (polarity)
- Try different batteries (test if batteries dead)

**Inaccurate readings**:
- Clean platform (remove debris, coffee grounds)
- Level surface (ensure scale on flat, stable counter)
- Tare before use (zero scale before weighing)
- Check max capacity (don't exceed 2000g)

**Timer not auto-starting**:
- Check sensitivity setting (Low = 5g threshold, may need Medium/High)
- Ensure MODE selected (auto-start only works in guided modes)
- Pour more decisively (slow, gentle pour may not trigger)
- Use manual timer (press TIMER button)

**Display hard to read**:
- Increase brightness (switch from dimmed to normal)
- Adjust viewing angle (LCD best straight-on)
- Clean display (wipe with soft cloth)
