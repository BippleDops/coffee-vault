## üéì Tips & Techniques

### Getting Best Results

**Accuracy Optimization**:
- **Placement**: Use on solid, level surface (not soft/uneven)
- **Stabilization**: Wait 2-3 seconds after placing object
- **Tare timing**: Tare after each addition for cumulative accuracy
- **Temperature**: Use in 15-30¬∞C environment (avoid extremes)
- **Airflow**: Avoid drafts, AC vents (can cause reading fluctuations)
- **Vibration**: Keep away from appliances, foot traffic

**Auto-Start Timer Tips**:
- **First use**: Test sensitivity levels with your setup
- **False triggers**: If frequent, switch to low sensitivity
- **Missed starts**: If common, switch to high sensitivity
- **Bloom timing**: First pour triggers timer (bloom starts at 0:00)
- **Manual override**: Press TIMER if auto-start fails
- **Consistent technique**: Pour same way each time for reliable triggering

**Battery Longevity**:
- **Auto-shutoff**: Enable 3-minute auto-shutoff (default)
- **Turn off**: Manually power off when done (don't rely on auto-shutoff)
- **Storage**: Charge to 50-70% for long-term storage
- **Temperature**: Store at room temperature (avoid heat)
- **Charge cycles**: Full discharge/charge cycles reduce lifespan
- **Partial charging**: Better to charge 20%‚Üí80% than 0%‚Üí100%

**Display Readability**:
- **Lighting**: Best in normal indoor lighting
- **Angle**: View straight-on (LED dims at extreme angles)
- **Sunlight**: Rotate scale if display washes out
- **Contrast**: Red LED on black = good contrast in most conditions

### Advanced Techniques

**Ratio Brewing Mastery**:
- **Set custom ratios**: Hold RATIO + (0-9) for 1:10 through 1:19
- **Quick switch**: Save favorite ratio for instant recall
- **Dose flexibility**: Scale recalculates target based on coffee weight
- **Visual tracking**: Watch remaining water countdown
- **Consistency**: Same ratio = repeatable results

**Pour Rate Control**:
- **Enable pour rate mode**: Press MODE to "RATE"
- **Target rate**: 2-3 g/s for most pour-overs
- **Visual feedback**: Bar graph shows if pouring too fast/slow
- **Practice**: Use pour rate mode to develop muscle memory
- **Competition prep**: Train consistent pour speed

**Multi-Stage Brewing**:
1. **Bloom**: Pour 2√ó coffee weight, note time
2. **Manual lap**: Mentally note bloom end time
3. **First pour**: Continue to 60% total water
4. **Second pour**: Finish to 100% target
5. **Drawdown**: Note total time when finished
6. **Record**: Log stages manually (no auto lap function)

**Troubleshooting Common Issues**:

**Scale won't turn on**:
- Check battery (charge 2-3 hours)
- Hold power button 3-5 seconds
- Try charging cable in different USB port
- Contact support if still unresponsive

**Inaccurate readings**:
- Check surface is level, solid
- Remove any debris from platform
- Tare before each use
- Test with known weight (calibration weight)
- Factory calibration may be needed (contact Brewista)

**Timer not auto-starting**:
- Check auto-start is enabled
- Adjust sensitivity level (low/med/high)
- Ensure pouring >0.5g (high) or >2.0g (low)
- Try manual timer start (TIMER button)
- Disable/re-enable auto-start

**Display issues**:
- Check brightness (not adjustable, single level)
- Clean display window with soft cloth
- Avoid direct sunlight (washes out LED)
- Restart scale (power off/on)

**Battery draining quickly**:
- Enable auto-shutoff (3-minute setting)
- Turn off manually when done
- Check for firmware updates (contact Brewista)
- Battery may be degraded (2-3 years) - consider replacement

**USB-C not charging**:
- Try different USB-C cable
- Try different power source (wall adapter, computer)
- Check port for debris (clean gently)
- Ensure cable fully inserted (reversible connector)
- Port may be damaged (warranty repair)
