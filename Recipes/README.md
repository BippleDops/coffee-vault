# Recipes Folder

This folder contains **Recipe Profiles** (`type: recipe-profile`) - reusable brewing recipes with success tracking and performance metrics.

## Purpose

Save and reuse successful brewing parameters:
- Reusable brewing recipes
- Method-specific recipe libraries
- Recipe variations and adjustments
- Success tracking and statistics
- Recipe sharing and documentation

## Usage

Create recipes using: `Templates/Recipe Profile.md`

## File Naming

Use descriptive names: `method-name-recipe-name.md` or `recipe-name.md`

## Linking

Link recipes to:
- Coffee Logs (`derived-from-log` property)
- Bean Profiles (best bean matches)
- Equipment Profiles (required equipment)
- Brewing Guides (related methods)
